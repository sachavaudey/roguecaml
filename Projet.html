Récit de réalisation

Voici, en détails, l'histoire de ce projet

C'est après de longues heures de réflexion, que l'idée du projet naît enfin : réaliser un jeu type Roguelike, dans lequel le personnage principal devrait résoudre des quêtes et des énigmes pour terminer le jeu.
Après quelques jours, un nom est posé sur le jeu :  Rogue-CAML.

Le pitch est simple :
" Le cri subit la plus grosse panne informatique de son histoire. En tant qu'ingénieur, vous êtes dépêché sur place pour tenter de la résoudre. Pour cela, vous allez devoir entrer dans les entrailles des machines pour trouver ce qui s'y trouve. Allez-vous y arriver ?"

A partir de là, la machine est lancée. Le développement commence dès février avec un premier niveau. En parallèle, le multijoueurs est implémenté et les premiers ennemis débarques.

Enfin, nous compilons le projet et c'est la version que nous pouvons vous présenter aujourd'hui.
